Introduction

What is an API?

API stands for Application Programming Interface. It is essentially a set of routines, protocols, and tools that can be used by programmers to build software applications. APIs allow developers to use the information avaliable on an exisiting database to incorporate into their programs.

Artsy.net

The Artsy website catalogues the world's art, consisting of both modern and contemporary works and related information. The site contains 500,000 images of art, architecture and design by 50,000 artists, providing a Pandora (for music) and Netflix (for film)-like art discovery service. Not only is this site changing the way art is discovered, it is also changing how art is bought and sold by providing educational content and details for galleries to make it easier for collectors to buy the art.

The Artsy API

The Artsy API provides access to the datbase of artworks, artists and related information avaliable on the artsy.net website. Follow this link to get to the official site containing the Artsy API documentation.
