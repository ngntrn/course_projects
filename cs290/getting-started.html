Getting Started

Signing Up

Before you can start using the API to make calls, you will first need to sign up for an Artsy developer account. Please follow the following link to do so: artsy.net/sign_up.

Once you're all signed in, you are now able to obtain an authentication token that can be used to make API requests by POSTing to /api/tokens/xapp_token. This is done using curl. The client_id and client_secret codes are given to you once you make a new app. Fill in the given IDs and run the following using curl: 

curl -v -X POST "https://api.artsy.net/api/tokens/xapp_token?client_id=XXXX&client_secret=XXXX"
The following JSON will be returned with your token: 
{
"type": "xapp_token",
 "token": "YOUR_TOKEN",
 "expires_at": "EXPIRATION_DATE_AND_TIME",
 "_links": {
 }
 }
This token can then be used to make API requests. I will show you how to do this in the next section.
