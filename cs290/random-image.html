Get a Random Image

API Request

Next, we will use the skills we have learned to obtain a random image from the Artsy database. This can be done by passing in a parameter "sample" to get a random element in the colletion. For example, "https://api.artsy.net/api/artists?key=id&sample=1" will return the JSON document for a random artist. An example below:

Each time "https://api.artsy.net/api/artists?key=id&sample=1", a different artist is returned. In the same way, we can also use "https://api.artsy.net/api/artworks?key=artist_id=&sample=1" to obtain random artworks.
