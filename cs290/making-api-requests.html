Making API Requests

Next Step

Now that you have obtained a token, we can begin making API requests. In the next example, I will show you how to make an API call to obtain the information about the artist Vincent Van Gogh. Using your token, perform the following curl command: 

curl -v -L https://api.artsy.net/api/artists/vincent-van-gogh -H 'X-Xapp-Token:YOUR_TOKEN'
